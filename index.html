Smartwatch App

Project Details:
For this project, I had to create a rapid mock-up of a fitness app for a smartwatch.

Project Requirements:

  • Be an app on a high-end smart watch (Pebble, Apple Watch, Samsung Gear). Pick one kind of smart watch that you want to use and stick with it, rather than trying to show what it would look like on all three.
  • Allow the user to start and stop a workout.
  • Allow the user to view how many calories they've burned, distance travelled, low/average/high heart rate so far during the current workout.
  • Allow the user to look through previous workouts and view the stats (listed above: calories burned, distance travelled, low/average/high heart rate) from each one.
  • One additional feature.
  • Date: Spring 2017
